# 🏡 Washington (King County) House Sales Dashboard | Tableau

## 📌 Overview

This project provides a comprehensive analysis of house sales data in Washington (King County) using interactive Tableau visualizations. It helps in understanding pricing trends, property distribution, and key factors influencing house sales.

## 🚀 Key Features

* 🏠 House price distribution analysis
* 📍 Geographic visualization of property prices
* 🛏️ Bedrooms & 🛁 bathrooms distribution
* 📈 Daily average house price trends
* 🎛️ Interactive filters (date, sqft, etc.)

---

## 💡 Key Insights

* House prices vary significantly by location
* Majority homes have 3–4 bedrooms
* Price trends fluctuate over time
* Property size strongly influences pricing
